Hi there !

Thank you for visiting gpx.studio, I am a 23y/o geeky cyclist and runner. I developed this tool in my spare time after messing with GPX files for my personal use for some time.

You can use it to :
  • Import GPX files
  • Draw new routes
    (be careful this tool is not intended to create the best cycle routes with no gravel)
  • Edit traces
    • Move, insert and delete points
      (including correct elevation data and preserving the average speed)
    • Cut from the start or the end
    • Change the starting time of the activity
    • Change the average speed of the activity
  • Combine and export multiple traces in the chosen order and respecting time precedence constraints (if any time data)

It is your own responsability not to use this website for digital doping purposes.

If you like the tool and use it frequently, please consider donating as to cover bills from API calls (map display, routing features and elevation requests) and encourage me to keep improving the website.

The project source code is available on GitHub where I also put future developments. Feel free to report any issues or send suggestions at contact.gpxstudio@gmail.com.

See you on the roads !
