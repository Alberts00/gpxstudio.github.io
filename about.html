About

gpx.studio is a free online GPX editor which allows to :

  • visualize multiple traces in different colors and aided by many different maps (cycling, hiking, satellite, Strava heatmap overlay, etc)
  • edit traces by adding, inserting, moving or deleting points
  • add timestamps to a trace
  • merge multiple traces or export them seperately, extending time, heart rate, cadence and temperature data where needed
  • save the result on your computer or to your Google Drive to get a shareable link

If you like the tool and use it frequently, please consider donating as to cover bills from API calls (map display, routing and elevation features) and encourage me to keep improving the website.

The project source code is available on GitHub where I also put future developments. Feel free to report any issues or send suggestions at contact.gpxstudio@gmail.com.
